## Sobre Mim

A jornada começou com desenvolvimento web como laboratório de aprendizado. HTML, CSS e JavaScript foram os primeiros módulos, mas o meu foco sempre foi entender como sistemas funcionam por trás da interface.

Hoje, meu trabalho é construir projetos que servem como veículos de aprendizado real: arquitetura, fluxo de dados, segurança e escalabilidade. Cada resultado é desenhado para funcionar como um passo concreto em direção a Software Engineering.

Não sou apenas um criador de páginas. Sou alguém que usa web e código como ferramentas para projetar e amadurecer sistemas reais, com um olhar crescente para Cybersecurity.

## Engineering Mindset

- Resolver problemas com clareza em vez de estética vazia.
- Entender a estrutura de sistemas e suas dependências.
- Construir soluções escaláveis com foco em manutenção.
- Usar projetos como motores de aprendizado contínuo.
- Priorizar engenharia sobre aparências.
- Transformar complexidade em resultados tangíveis.

### Roadmap discreto

HTML → CSS → JavaScript → React → Python → C# → Software Engineering → Cybersecurity

## Mission

> Become a Software Engineer

- Goal: projetar e construir sistemas com disciplina.
